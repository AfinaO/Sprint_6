Все локаторы находятся в директории locators.
Локаторы для главной страницы и вопросов находятся в файле main_page_locators.py
Локаторы для страницы заказа находятся в файле order_page_locators.py

В директории page_objects находятся файлы с классами для каждой тестируемой страницы.
Объекты для всех страниц находятся в файле base_page.py
Все объекты для главной страницы находятся в файле main_page.py
Объекты для страницы заказа находятся в файле order_page.py

В директории test находятся файлы с тестами, для каждой страницы — свой файл.
Тесты в файле test_main_page.py проверяют соответствие текста раскрывающихся ответов в разделе "Вопросы о важном" на главной странице приложения.
Тесты в файле test_navigation.py проверяют переходы на главную страницу сервиса и на главную страницу "Дзена" при клике по разным частям логотипа в шапке.
Тест в файле test_order_page.py покрывает позитивные сценарии оформления заказа. В тесты через параметризацию передаются два набора пользовательских данных и две точки входа.

Файл conftest.py содержит фикстуру, необходимую для запуска тестов.

Файл constants.py содержит ссылку на сайт.

Директория allure_results содержит JSON-файлы с результатами выполнения тестов для генерации отчета.

Файл testdata.py хранит тестовые данные полей ввода.

В файле requirements.txt перечислены все внешние зависимости исполняемых тестов для удобной установки одной командой.
